## 插件说明 ##

 - 版本: v1.0.5
 - 作者: [冰剑](https://github.com/binjoo)
 - 主页: <https://github.com/binjoo/SlimBox2-for-Typecho>

## 使用方法 ##

 1. 下载插件
 2. 如果安装有老版本，请先卸载老版本，再删除插件文件
 3. 将插件上传到 /usr/plugins/ 这个目录下
 4. 登陆后台，在“控制台”下拉菜单中进入“插件管理”
 5. 启用当前插件
